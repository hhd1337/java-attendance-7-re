# woowacourse 8기 최종 코딩테스트

## 기능구현 목록

### 메뉴 출력/분기/반복/종료 기능

- [x] 기능 메뉴를 출력한다.
- [x] 메뉴의 기능 하나가 끝나면 곧바로 기능목록 메뉴가 다시 출력된다. (Q를 입력 시 종료된다.)
- [x] 메뉴 입력을 받는다.
    - 예외) 메뉴에 없는 문자열일 경우
    - 예외) 주말 또는 공휴일에 출석을 확인하거나 수정하는 경우

### 파일을 읽어 출석부를 초기화하는 기능

- [x] src/main/resources/attendances.csv 파일을 읽어 출석기록을 로드한다.

### 1. 출석확인 기능

- [x] 닉네임을 입력받는다.
    - 예외) 닉네임이 출석부에 없는 경우
    - 예외) 이미 해당 닉네임의 크루가 출석을 하였는데 다시 출석 확인을 하는 경우
- [x] 등교시간을 입력받는다.
    - 예외) 등교 시간이 캠퍼스 운영 시간이 아닌 경우 (추후 구현)
- [ ] 출석부에 저장한다.
- [ ] 출석 기록을 출력한다.

### 2. 출석수정 기능

- [ ] 출석을 수정하려는 크루의 닉네임을 입력받는다.
    - 예외) 닉네임이 출석부에 없는 경우
- [ ] 수정하려는 날짜(일)을 입력받는다.
    - 예외) 미래 날짜로 출석을 수정하는 경우
    - 예외) 수정하려는 날짜에 출석기록이 없는 경우
- [ ] 수정하려는 시간을 입력받는다.
- [ ] 원래 출석상태와 바뀐 출석상태를 포함하여 수정 결과를 출력한다.

### 3. 크루별 출석 기록 확인 기능

- [ ] 출석을 수정하려는 크루의 닉네임을 입력받는다.
    - 예외) 닉네임이 출석부에 없는 경우
- [ ] 날짜별 출석기록을 출력한다.
    - 조건) 아예 등교를 하지 않아 결석인 경우는 시간을 "--:--" 라고 출력한다.
    - 조건) 공휴일이나 주말은 아예 출력하지 않는다.
- [ ] 이번달 해당 크루의 출석횟수와 지각횟수, 결석횟수를 출력한다.
- [ ] 경고/면담/제적 대상자라면, "__ 대상자입니다." 라고 출력한다.

### 4. 제적 위험자 조회 기능

- [ ] 전날까지의 크루 출석 기록을 바탕으로 제적 위험자들을 파악한다.
- [ ] 제적 대상자, 면담 대상자, 경고 대상자순으로 정렬한다.
- [ ] 각 제적,면담,경고 대상자 중에서는 지각을 결석으로 간주하여 내림차순 정렬한다.
- [ ] 정렬된 제적 위험자들을 출력한다.
